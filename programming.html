PROGRAMMING LANGUAGES

WHAT IS PROGRAMMING?

Programming is the process of giving instructions to a computer to perform specific tasks. These instructions are written using programming languages.


WHY PROGRAMMING IS IMPORTANT?

Programming helps in problem solving, automation, and software development. It is an essential tool for computer science students.


TYPES OF PROGRAMMING LANGUAGES

  • High Level Language
  • Low Level Language

HIGH LEVEL LANGUAGE

These are designed to be user-oriented. They use English-like words which makes them easy for us to read and write. They are very easy to learn and debug.

EXAMPLES OF PROGRAMMING LANGUAGES

LANGUAGE USE
C++ Used for learning Programming Fundamentals
Python Used for General-Purpose Programming
Java Used for desktop and mobile applications
JavaScript Used for Web Development

LOW LEVEL LANGUAGE

These are machine-oriented. They deal directly with the computer's physical components like memory addresses and registers. There are two types of Low Level Languages.

  • Machine Language
  • Assembly Language

Machine Language

This is the lowest level of programming language. It is the only language that the computer's processor (CPU) can understand directly without any translation. It consists entirely of binary code (0s and 1s).

Assembly Language

This is a small step up from machine code. It replaces the confusing strings of 1s and 0s with short, English-like "nicknames" called mnemonics.
